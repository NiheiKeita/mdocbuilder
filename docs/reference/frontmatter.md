# フロントマター設定

このページは frontmatter をあえて付けず、最初の h1 がタイトルとして採用されるパターンを確認するためのサンプルです。

## 代表的なキー

| キー | 役割 |
| --- | --- |
| title | ページタイトルとして利用 |
| description | 一覧や meta description に利用 |
| updatedAt | 必要なら build 側で補完可能 |

## 例

```yaml
---
title: 検索を試す
description: 検索UIの確認用ページ
---
```
